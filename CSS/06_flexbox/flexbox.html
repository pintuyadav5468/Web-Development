P
Parent (Container) Properties
Applied on the flex container element
🖥
[1] display
Activates flexbox on the container element.
flex inline-flex
→
[2] flex-direction
Sets the main axis — the direction items flow.
row row-reverse column column-reverse
↩
[3] flex-wrap
Controls whether items wrap to new lines.
nowrap wrap wrap-reverse
⚡
[4] flex-flow
Shorthand for flex-direction + flex-wrap combined.
row nowrap row wrap column wrap column-reverse wrap
⟺
[5] justify-content
Aligns items along the main axis.
flex-start flex-end center space-between space-around space-evenly
⟸
[6] align-items
Aligns items along the cross axis (single line).
stretch flex-start flex-end center baseline
☰
[7] align-content
Aligns multiple lines along the cross axis.
normal flex-start flex-end center space-between space-around stretch
⊞
[8] gap
Sets spacing between items (not on outer edges).
0 16px 20px 10px row-gap column-gap
C
Child (Item) Properties
Applied on individual flex items
①
[1] order
Controls the visual display order of items.
0 -1 1 2 any integer
↔
[2] flex-grow
How much the item grows to fill free space.
0 1 2 0.5
↔
[3] flex-shrink
How much the item shrinks when space is tight.
1 0 2 0.5
📐
[4] flex-basis
Initial size before growing or shrinking.
auto 0 200px 50% content
⚡
[5] flex
Shorthand: flex-grow + flex-shrink + flex-basis.
0 1 auto 1 auto none 2 1 100px
⟸
[6] align-self
Overrides align-items for this single item only.
auto flex-start flex-end center baseline stretch

Cheat Sheet
All properties at a glance
# Property Applied On Axis Default Common Values
[1] display Parent — block flex, inline-flex
[2] flex-direction Parent Main row row, column, row-reverse, column-reverse
[3] flex-wrap Parent Cross nowrap nowrap, wrap, wrap-reverse
[4] flex-flow Parent Both row nowrap Shorthand for direction + wrap
[5] justify-content Parent Main flex-start center, space-between, space-evenly
[6] align-items Parent Cross stretch center, flex-start, flex-end, baseline
[7] align-content Parent Cross normal center, space-between (multi-line only)
[8] gap Parent Both 0 16px, 20px 10px, row-gap, column-gap
[1] order Child — 0 -1, 0, 1, 2 (lower = first)
[2] flex-grow Child Main 0 0 (no grow), 1 (grow), 2 (grow 2×)
[3] flex-shrink Child Main 1 0 (no shrink), 1 (shrink), 2 (shrink 2×)
[4] flex-basis Child Main auto 0, 200px, 50%, content
[5] flex Child Main 0 1 auto 1, auto, none, 2 1 100px
[6] align-self Child Cross auto center, flex-start, flex-end, stretch
</>

flexbox-complete.css 
/* ── PARENT PROPERTIES ── */

.container {
  /* [1] display */
  display: flex;
  display: inline-flex;

  /* [2] flex-direction */
  flex-direction: row;
  flex-direction: row-reverse;
  flex-direction: column;
  flex-direction: column-reverse;

  /* [3] flex-wrap */
  flex-wrap: nowrap;
  flex-wrap: wrap;
  flex-wrap: wrap-reverse;

  /* [4] flex-flow (shorthand) */
  flex-flow: row nowrap;
  flex-flow: column wrap;

  /* [5] justify-content */
  justify-content: flex-start;
  justify-content: flex-end;
  justify-content: center;
  justify-content: space-between;
  justify-content: space-around;
  justify-content: space-evenly;

  /* [6] align-items */
  align-items: stretch;
  align-items: flex-start;
  align-items: flex-end;
  align-items: center;
  align-items: baseline;

  /* [7] align-content */
  align-content: normal;
  align-content: flex-start;
  align-content: flex-end;
  align-content: center;
  align-content: space-between;
  align-content: stretch;

  /* [8] gap */
  gap: 16px;
  gap: 20px 10px;
  row-gap: 20px;
  column-gap: 10px;
}

/* ── CHILD PROPERTIES ── */

.item {
  /* [1] order */
  order: 0;
  order: -1;
  order: 1;

  /* [2] flex-grow */
  flex-grow: 0;
  flex-grow: 1;
  flex-grow: 2;

  /* [3] flex-shrink */
  flex-shrink: 1;
  flex-shrink: 0;
  flex-shrink: 2;

  /* [4] flex-basis */
  flex-basis: auto;
  flex-basis: 0;
  flex-basis: 200px;
  flex-basis: 50%;

  /* [5] flex (shorthand) */
  flex: 0 1 auto;
  flex: 1;
  flex: auto;
  flex: none;

  /* [6] align-self */
  align-self: auto;
  align-self: flex-start;
  align-self: flex-end;
  align-self: center;
  align-self: stretch;
  align-self: baseline;
}

/* ── PATTERNS ── */

.center        { display:flex; justify-content:center; align-items:center; }
.equal-cols > * { flex:1; }
.sticky-footer { display:flex; flex-direction:column; min-height:100vh; }
.sticky-footer main { flex:1; }
.card-grid > * { flex:1 1 260px; }
.push-last > *:last-child { margin-left:auto; }
